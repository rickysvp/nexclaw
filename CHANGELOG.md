## [0.2.0] - 2026-02-28

### Added
- 创建钱包详情页面 (`/dashboard/wallet/[id]`)
  - 显示完整钱包信息和交易统计
  - 交易记录列表
  - 导出钱包数据功能
  - 安全评分显示
- 实现钱包编辑功能
  - 编辑钱包名称和描述
  - 表单验证和保存
- 实现钱包删除功能
  - 删除确认对话框
  - 必须输入 "DELETE" 确认
  - 安全警告提示
- 创建安全规则配置页面 (`/dashboard/security`)
  - PIN 码设置/修改
  - 交易限额配置（单笔/日累计）
  - 地址白名单管理
  - 合约黑名单管理
  - 自动计算安全评分
- 实现转账功能（深度交互）
  - 多步骤转账流程
  - 地址格式验证
  - 余额检查（含 Gas 费）
  - 黑名单检查
  - 交易限额检查
  - 白名单警告
  - PIN 码验证
  - 处理中状态
  - 成功/失败结果
- 实现充值功能
  - 钱包地址展示
  - 一键复制
  - QR 码生成
  - 网络信息
  - 充值说明
- 更新导航栏，添加安全规则链接

### Changed
- 扩展 Wallet 类型，支持 name 和 description
- 更新 Mock 数据，添加安全规则
- 优化 UI 交互和动画效果

## [0.1.0] - 2026-02-28

### Added
- 初始化项目架构
- 创建首页，包含 Hero、特性展示、定价方案、用户评价
- 创建仪表盘页面，支持钱包管理、交易历史、小龙虾测试
- 创建关于我们页面，展示公司故事、价值观、团队
- 创建文档中心，包含快速开始、API 参考、FAQ
- 创建联系我们页面，包含联系表单和信息
- 创建隐私政策和服务条款页面
- 实现响应式设计，支持桌面和移动端
- 添加 Framer Motion 动画效果
- 使用 Tailwind CSS + shadcn/ui 组件库
- 创建项目需求文档 (PRD)

### Features
- **零助记词设计**：嵌入式 AA 钱包技术
- **Agent 友好**：通过 UID 轻松集成
- **3 秒创建**：快速创建 Agent Wallet
- **安全规则**：PIN 码、交易限额、白名单/黑名单
- **交易历史**：查看和管理所有交易记录
- **小龙虾测试**：模拟 Agent 调用测试工具

## 版本号说明

版本号格式：主版本号.次版本号.修订号

- **主版本号**：重大功能变更或架构调整
- **次版本号**：新增功能或重大改进
- **修订号**：Bug 修复或小优化

## 版本发布流程

1. 更新 VERSION 文件
2. 更新 CHANGELOG.md
3. 创建 Git 标签：`git tag -a v0.1.0 -m "版本 0.1.0"`
4. 推送标签：`git push origin v0.1.0`
5. 在 GitHub 上创建 Release
