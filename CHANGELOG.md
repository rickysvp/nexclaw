## [0.1.0] - 2026-02-28

### Added
- 初始化项目架构
- 创建首页，包含 Hero、特性展示、定价方案、用户评价
- 创建仪表盘页面，支持钱包管理、交易历史、小龙虾测试
- 创建关于我们页面，展示公司故事、价值观、团队
- 创建文档中心，包含快速开始、API 参考、FAQ
- 创建联系我们页面，包含联系表单和信息
- 创建隐私政策和服务条款页面
- 实现响应式设计，支持桌面和移动端
- 添加 Framer Motion 动画效果
- 使用 Tailwind CSS + shadcn/ui 组件库
- 创建项目需求文档 (PRD)

### Features
- **零助记词设计**：嵌入式 AA 钱包技术
- **Agent 友好**：通过 UID 轻松集成
- **3 秒创建**：快速创建 Agent Wallet
- **安全规则**：PIN 码、交易限额、白名单/黑名单
- **交易历史**：查看和管理所有交易记录
- **小龙虾测试**：模拟 Agent 调用测试工具

## 版本号说明

版本号格式：主版本号.次版本号.修订号

- **主版本号**：重大功能变更或架构调整
- **次版本号**：新增功能或重大改进
- **修订号**：Bug 修复或小优化

## 版本发布流程

1. 更新 VERSION 文件
2. 更新 CHANGELOG.md
3. 创建 Git 标签：`git tag -a v0.1.0 -m "版本 0.1.0"`
4. 推送标签：`git push origin v0.1.0`
5. 在 GitHub 上创建 Release
